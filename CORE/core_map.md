# CORE/
🧠 Cerveau de l’application ONE  
Contient la logique centrale commune à toutes les matières.
- **ci.py**  
    Paramètres globaux, styles et fonctions utilitaires communes.
- **core_map.md**  
    Carte et documentation du dossier `ONE/CORE/`.
- **dataloader/**  
    Gestion du chargement des données depuis `ONE/DATA/`.
- **exercise.py**  
    Classe centrale reliant `main.py` aux différentes matières  
    situées dans `ONE/MATIERE/`.

## CLASSE_SECONDAIRE/
Classes secondaires, stables ou peu utilisées.

- **classe_secondaire_map.md**
- **couleur**
